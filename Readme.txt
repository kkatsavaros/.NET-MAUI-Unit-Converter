.NET MAUI - Unit Converter



This app is a complete Unit Converter for information, length, volume, mass, temperature, energy, area, speed and duration.


For all that units conversion i use thw UnitsNet NuGet package which gets all common units of measurement and converts between them.


Also for the implementation of INotifyPropertyChange interface χρησιμοποιώ το PropertyChanged.Fody NuGet Package.
